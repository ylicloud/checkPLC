# 语音词片目录

放置预录词片（优先 `.wav`，也支持 `.mp3`）：

| 文件名 | 内容 |
|--------|------|
| `1` … `32` | 整段通道号（21→「二一」），**优先使用，延迟最低** |
| `ma4` … `ma20` | 整段「四毫安」…「二十毫安」 |
| `0` … `10`、`shi`、`haoan` | 拼接兜底 |
| tongdao / dian | 可选词 |

生成：

```bash
pip install edge-tts
python scripts/generate_wavs.py
```

若尚无词片，页面会自动回退到浏览器中文 TTS（延迟明显更高）。
